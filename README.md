# CIPHER v3 🤖

**Competitive Intelligence Platform** - AI-powered анализ конкурентной среды для маркетологов и product-менеджеров.

## 📊 Описание

CIPHER v3 - это интерактивная платформа для глубокого анализа конкурентов. Ответь на 7 вопросов о своём продукте, и получи:

✅ **8 табов анализа** с таблицами данных TOP 5 конкурентов  
✅ **Стратегия на год** с разбивкой по кварталам  
✅ **"Учись у" рекомендации** - чей опыт перенять в каждой области  
✅ **Финансовый прогноз** - инвестиции, ARR, ROI  
✅ **Конкретные действия** - не шаблоны, а привязка к вашему контексту  

## 🎯 Функционал

### 📋 Форма вопросов (Степпер)
- Вопрос за вопросом для удобного заполнения
- NOVA (AI ассистент) слева дает подсказки для каждого вопроса
- Валидация - нельзя пройти без ответа
- Поддержка input, textarea, radio, multi-select

## 🚀 Как использовать

### Локально

1. Скачай `CIPHER_v3_FINAL.html`
2. Открой в браузере
3. Заполни 7 вопросов про свой продукт
4. Получи анализ конкурентов и стратегию

### Online (GitHub Pages)

```bash
# 1. Создай репозиторий на GitHub с именем: CIPHER-v3
# 2. Клонируй его
git clone https://github.com/ТВОЙ_ЮЗЕР/CIPHER-v3.git
cd CIPHER-v3

# 3. Добавь файлы
cp CIPHER_v3_FINAL.html ./index.html
git add .
git commit -m "Initial commit: CIPHER v3"
git push origin main

# 4. Включи GitHub Pages в настройках репозитория
# Перейди на Settings → Pages → Source: main branch
# Ссылка: https://ТВОЙ_ЮЗЕР.github.io/CIPHER-v3
```
