# МАРГО — Marketing Research & Growth Optimizer

Отдельный маркетинг-агент для анализа конкурентов.
Работает параллельно с основным Cipher-анализом.

## Структура

```
Margo/
├── index.html          — главный экран агента
├── api/
│   ├── scrape.js       — скрейпинг сайтов, G2, Trustpilot
│   ├── ads.js          — анализ рекламы (Facebook Ad Library)
│   └── margo.js        — AI-агент МАРГО (диалог)
├── netlify/
│   └── functions/      — serverless версии API
└── README.md
```

## Модули

1. **Сайт конкурентов** — структура лендинга, UVP, CTA, воронка
2. **Отзывы клиентов** — G2, Trustpilot, что хвалят / на что жалуются
3. **Реклама и таргет** — Facebook Ad Library, форматы, офферы, боли
4. **МАРГО** — диалог на основе собранных данных, контент-план
