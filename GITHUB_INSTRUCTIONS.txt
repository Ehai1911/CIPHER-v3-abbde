3️⃣ ЗАГРУЗИ ФАЙЛЫ НА GITHUB (ВАРИАНТ A - ЧЕРЕЗ БРАУЗЕР)

На странице репозитория:
- Нажми "Add file" → "Upload files"
- Перетащи CIPHER_v3_FINAL.html
- Перетащи README.md
- Нажми "Commit changes"

3️⃣ ЗАГРУЗИ ФАЙЛЫ НА GITHUB (ВАРИАНТ B - ЧЕРЕЗ GIT ТЕРМИНАЛ)

Если у тебя установлен Git:

# 1. Клонируй репозиторий
git clone https://github.com/ТВОЙ_ЮЗЕР/CIPHER-v3.git
cd CIPHER-v3

# 2. Положи файлы в папку (скопируй туда CIPHER_v3_FINAL.html и README.md)

# 3. Переименуй HTML в index.html (чтоб открывалась сразу)
mv CIPHER_v3_FINAL.html index.html

# 4. Добавь файлы в Git
git add .
git commit -m "🚀 Initial commit: CIPHER v3 - Competitive Intelligence Platform"
git push origin main

✅ ГОТОВО! Файлы в GitHub.
